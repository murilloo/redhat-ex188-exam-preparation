List container
$ podman ps 
$ podman ps --format='{{.Names}}'

List file inside a container
$ podman exec mycontainer ls /etc/hosts

Copy file from a container to local
$ podman cp mycontainer:/etc/hosts hosts

Create a network
$ podman network create mynetwork

Start a container
$ podman run -d --name mycontainer-backend --net mynetwork -p 8080:8080 https://quay.io/repository/murill00/mycontainer-backend

Copy file to a container
$ podman cp hosts mycontainer:/tmp

Start a new container
$ podman run -d --name mycontainer2 --net mynetwork https://quay.io/repository/murill00/mycontainer-client

Verify network
$ podman inspect mynetwork

Check backend from client
$ podman exec mycontainer2 curl -s http://mycontainer:8080
